# RailwayMS — Modern Railway Booking System

Developed by **Prathamesh Giri**

This project is developed for open source.

## Overview
RailwayMS is a premium, fully responsive railway ticket booking application built with React, Vite, Redux Toolkit, and Tailwind CSS. It features a modern, animated user interface with seamless user experiences ranging from searching trains, selecting seats, and managing passenger details, to a mock payment gateway and admin dashboard.
